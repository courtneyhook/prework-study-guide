HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Headings are structured 1-6 with h1 being the main heading on the page
  • There should only be one h1 heading on each page--it is the most important heading
  • HTML is the structure of a webpage

CSS

  • CSS adds style to a webpage
  • You can style more than one elements by using a comma
  • CSS can change colors, add borders, add shadows and much more
  • The CSS Rule is use a selector to say what structure you want to style and a declaration. A declaration tells the property to change (:) and a value to change it to.
  • The margin tells how much space we want around the outside of an element
  • The padding tells how much space we want around the content inside an element
